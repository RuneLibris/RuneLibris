Overview[edit]

A build (sometimes refered as a revision) is a term used when talking about a specific version of RuneScape[1].

Popular builds[edit]

The following is a list of popular/notable game build within the private server community[2]:

Build number Date Notes
194 07 January 2004[3] RuneScape 2 beta[4]
RuneScape 2 releases on 29 March 2004[5]
225 18 May 2004[6] None
317 18 June 2005[7] None
435 30 October 2006[8] None
448 08 February 2007[9] None
RuneScape HD releases on 01 July 2008[10]
530 28 January 2009[11] None
667 13 September 2011[12] None
718 29 May 2012[13] None
742 07 November 2012[14] None
RuneScape 3 releases on 22 July 2023[15]
830 07 January 2015[16] None
876 08 December 2016[17] None
910 11 December 2019[18] None
Java client is discontinued on 18 December 2019[19]
