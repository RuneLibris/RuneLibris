Overview[edit]

This page host a list of source along with information on how reliable they are[1]

Source reliability[edit]

Tier 1[edit]

Source Notes
Screenshots & videos of RS3, OSRS and RSC It should be noted that some pre-2007 content in OSRS has been reworked/altered
Packet dumps Examples includes RSC+ replays and output of RSProx.
Cache data None
RuneScript snippet & data dumps from Jmods None

Tier 2[edit]

Source Notes
RS Wiki, OSRS Wiki and RSC Wiki None
Fan sites None

Tier 3[edit]

Source Notes
Lost City Trusted RuneScape preservation project which actively document sources used[2]
OpenRSC Trusted RuneScape preservation project (in most case data should be identical to RSC+ replays)[3]

Tier 4[edit]

Source Notes
Vidyascape A closed-source RuneScape preservation project. It ranks this low as the developement process is not transparent. The accuracy of preserved data cannot be easily checked[4]

Tier 5[edit]

Source Notes
Other private servers Most private servers are known for taking liberties and generally not having a focus on RuneScape preservation. Only to be used when there is no other option[5]
