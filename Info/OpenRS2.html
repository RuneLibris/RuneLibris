Overview[edit]

OpenRS2 is an online archive (link) of caches[1].

Fetching an individual file[edit]

It is possible to fetch an individual .dat file of a given cache by using the API that follows the /caches/<scope>/<id>/archives/<archive>/groups/<group>.dat URL structure[2]. A concrete example being https://archive.openrs2.org/caches/runescape/2064/archives/7/groups/0.dat[3].
