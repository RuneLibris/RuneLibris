Overview[edit]

OSRS 196-3 examines refers to a dataset containing item, NPC and location examines for OSRS 196-3 (28 May 2021). It was originally published by Kris on Rune-Server. The version present in this archive had subject names added by manpaint[1].

Archive details[edit]

  1. The original dataset was downloaded from this Rune-Server thread[2].
  2. Build 196 subject definition were downloaded from this Rune-Server thread made by Jordan[3].
  3. The JSON parser was used to extract ID-name pairs (local archive)[4].
  4. The Python replacement script was used to insert subject names into Kris's examine dataset[5].
  5. Missing entries were filled using Abextm's OSRS cache viewer[6].
