Overview[edit]

Mod Nin NPC spawns 2020-07-04 refer to a set of NPCs spawns generated by Mod Nin for RuneScape Wiki users[1]

Context[edit]

The following is a transcript of the conversation (and therefore context) surrounding the dataset[2]


On 02 July 2020, BlackHawk pings Mod Shauny in the RuneScape Wiki Discord, hoping to be able to have updated NPCs coordinate. Mod Nin comes in and says he can generate a list of spawns: 

BlackHawk — 2020-07-02 16:18
@Shauny, reckon it would be possible to try to get some updated npc coordinates?

Mod Nin — 2020-07-03 10:56
^ I can look into this for you
[11:01]
Not sure how Shauny would have created that list, if someone could ping me the old list I'll try and make one in the same format

mejrs — 2020-07-03 11:12
I cant find it right now but it had the id, name and location
[11:12]
Of every named npc on mainland and anachronia
[11:12]
If you have this for every npc across the map that'd be awesome

A few messages later, Mod Nin then posts the following file:

It's a few months out-of-date because of some remote working limitations

Later on, BlackHawk makes the observation that some NPCs are missing: 

BlackHawk — 2020-07-03 15:38

[15:54]
Yeah it looks partial - Divination wisps are missing too

On 04 July 2020, Mod Nin posted the following: 

Mod Nin — 2020-07-04 07:33
Had a look at wisps specifically, they're not appearing on the list because they're spawned via script slightly after the server boots

BlackHawk — 2020-07-04 07:59
Thanks for looking! Are they the only NPC that function like that or are there others?

Mod Nin — 2020-07-04 08:01
Can only assume there's more
