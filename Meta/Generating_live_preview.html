Meta/Generating live preview

From RuneLibris
Jump to: navigation, search

Overview[edit]

This page contains instruction to generate the RuneLibris live preview[1].

Instruction[edit]

  • Create a spreadsheet with two columns named Pagename and URL
  • Go to this page and refresh the page.
  • Copy pages names in Pagename spreadsheet collumns
  • Copy all pages URL name using Copy Selected Links Firefox extension
  • Decode all URLs in Notepad++ (Plugins, MIME Tools, URL decode)
  • Copy URLs in URL spreadsheet collumns
  • Write a wget .bat script that follow this pattern:
wget "http://localhosthttps://runelibris.github.io/RuneLibris/Aspects/Build" --force-directories
wget "http://localhosthttps://runelibris.github.io/RuneLibris/Aspects/Cache" --force-directories
wget "http://localhosthttps://runelibris.github.io/RuneLibris/Aspects/Cache-client_pair" --force-directories
  • Run the script, wait for it to finish.
  • Navigate downloaded files subfolder until you reach Main page file
  • Move the downloaded files to C:\RULI_DEMO
  • In grepWin, enter the C:\RULI_DEMO path
  • Ensure that Include subfolders is checked
  • Ensure that All sizes is checked
  • In grepWin replace the following:
Find string Replace string 
https://runelibris.github.io/RuneLibris/
