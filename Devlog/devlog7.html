RuneLibris Devlog #7

The first thing I did this week was to fix the broken page links issue. In most cases, this was due to the presence of the ":" character on the page names for items, locs and NPCs. As such, I altered all of them with a script to remove the problematic characters. I also changed links in the pages' text.

The GitHub site is now properly usable. I am certain that there are still some edge cases somewhere on the wiki, but they will be ignored for now as this website is simply a demonstration. Even if a page cannot be accessed, the .html file should still be accessible through the GitHub repository.

The other thing I did this week was to finish the unlock hints page. As I suspected, RS Wiki had lots of outdated data. As such, I dumped the hints myself using Wireshark. The data was then compared with RS Wiki data and manually reviewed as to ensure that no mistakes were made.

I also created a template that will be used to signify when the subject of a given page is considered archived. It also includes a field that says which was the last game update the data account for. This will be useful should I ever decide to update those pages with new data.

I also figured out how to dump location configurations with Skillbert's cache tool. Details have been consigned to the tool's page on the wiki.

The next thing I intent to is to finish documenting the non-standard path for Tutorial Island 2018. This will allow me to minimize the amount of RuneScape account I will need to create in the future. Once that is done, I will look into documenting the music map for the RuneScape Surface.
