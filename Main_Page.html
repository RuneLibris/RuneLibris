Introduction[edit]

RuneLibris is an offline database that seek to document non-repeatable server interactions that existed for a significant amount of time in RuneScape (RS2, RSHD, RS3).

Content that was released after 2021-11-23 is considered low priority.

The current build number of the wiki is 37.
