Overview[edit]

468 (Summoning WIP) is the meta-name given to a dev build containing an unfinished version of the Summoning skill found in a Jagex tape backup by Mod Lykos[1]

Details[edit]

  • In this version, Summoning was called Animal Lore[2]
  • There is a familiar named Boar which could be summoned with 45 Animal Lore; it dosen't exist in the released version of the skill[3]
