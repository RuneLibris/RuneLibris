Notes[edit]

  • The cache only includes mapsquare have some kind of function rather than the whole game world. For example 18,58 do not have an entry. There are however some empty mapsquares included in the cache for reasons unknown[4]. An example of this is how there is empty mapsquare for 17,35 in build 317[5]
  • XTEAs file present on OpenRS2 do not includes missing mapsquares not covered by the XTEA keys (i.e 66,81 for 742)[6]
