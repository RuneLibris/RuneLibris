Overview[edit]

Doric's Quest is a quest that was removed on 17 October 2012, being replaced by What's Mine is Yours[1].

Quests states[edit]

The following is NOT a list of quest varbits states. For more information see the quest state policy.

Quest states are as follow (based on dialogue seen in [Roccorii]'s playthrough):

State Objective Notes
0 (Quest not started) Talk to Doric (ID 284) to start the quest None
1 Talk to Doric (ID 284) with 6x Clay (ID 434), 4x Copper ore (ID 436) and 2x Iron ore (ID 440) in inventory None
2 (Quest completed) N/A None
