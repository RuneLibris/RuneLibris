Overview[edit]

This pages docoment the behavior of the RuneScript command that happens when the player unlock a new listed music track[1].

RuneScript snippet[edit]

The following is a RuneScript from Old School RuneScape[2]:

~music_unlocksong(true,^music_ID_inferno);


Script[edit]

When Player unlock a new listed music track
  Music track is flagged as unlocked
  Interaction Dynamic message based on track name[3]
  Music track unlock count + 1
 
If Music track unlock count is equal or is above 500[4]
 


If Air Guitar emote is not unlocked[4]
  Air Guitar emote flagged as unlocked[4]
  You've unlocked 500 music tracks and can use a new emote![5] Fetch image[6]
end


end



end
