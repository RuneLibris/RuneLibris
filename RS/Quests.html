Overview[edit]

This page is an index and documentation of quest history in RuneScape[1].

Archive details[edit]

  1. Quest and mini-quest release date data was extracted from RuneScape Wiki using a DPL and then converted to RuneLibris format using LibreOffice and Notepad++[2].
  2. Using another RuneScape Wiki DPL, a list of removed quest and miniquests was gathered and then added as removal entries on this page[3].
  3. A list of quick chat quest names (dataset here) was extracted from a ClientScript dump of an unknown 2017 cache found in RuneLibris' 2023 archives. Said list was then compared to data gathered below to determine which quest have been renamed[4].
  4. Added removal dates for recent temporary holiday quests[5].
  5. Using intel gathered from looking at the archives of the RuneScape Wiki Discord, the 2019-04-15 update that renamed miniquests was documented[6].
  6. Tutorial information previously documented was converted to the new format and added to the table below[7].

Shortest F2P quests[edit]

Although the quest varbits cannot be used to measure the complexity of a given quest, it can be used to measure the simplicity of some of the earliest quests of the game - an information useful when running preservation tests. As such, the shortest quests in RuneScape 3 are as follows[11].

The "Minimum steps required" value is based on amounts of minimum amount of subjects you need to interface with to complete the quest. Data is based on RS Wiki's quick guides (unecessary data having being trimmed out). This assumes the player have required items before starting the quest.

Endvalue Quest name Minimum steps required
3 Ernest the Chicken 30
3 Witch's Potion 5
3 Vampyre Slayer 9
2 Imp Catcher (2017) 13
2 Cook's Assistant (2009) 9

As seen, the simplest quest in RuneScape 3 is Witch's Potion[11].

Quick chat anomaly[edit]

Player used to be able to mentions quest titles in Quick chat (which were stored in clientScript 1541) but some strings differ from their standard counterparts. While for most quest this either simply the absence or addition of the string The. There are however two quests that stands out[12]:

String Actual title
Mourning's End I Mourning's End Part I
Mourning's End II - Temple of Light Mourning's End Part II
